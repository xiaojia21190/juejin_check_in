# juejin_check_in

## 介绍

这是一个用于自动签到的脚本,可以实现每天自动签到,并且可以获取每天的签到奖励。

## 使用方法

1. 打开浏览器,访问 https://juejin.cn/
2. 点击右上角的头像,选择个人中心
3. 点击右上角的签到按钮
4. 等待签到成功,即可获取签到奖励
5. 在浏览器中，点击 EditThisCookie 登陆后，点击导出cookie


# 第二步
在以下内容中，填入cookie

```javascript
  let cookieArray = [];
```

# 第三步
```bash
  npm install
  npm run start
```
